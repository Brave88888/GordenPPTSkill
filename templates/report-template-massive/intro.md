# 118 页超大工作汇报合辑 · 蓝色

## 一句话
深蓝主调的"职场汇报百科全书"，118 页**几乎覆盖所有职场场景**：工作汇报、岗位竞聘、述职、项目复盘、财务/营销/HR 数据看板、思维模型（SWOT/PDCA/KISS/5W1H/鱼骨/漏斗/金字塔/树状/甘特/齿轮）等等，"挑一份就是一份完整 PPT"。

## 风格标签
- 整体风格：**蓝色商务 / 信息密集 / 全场景**
- 排版复杂度：**很高**（每页 10-30+ 文字块）
- 视觉元素：深蓝矩形 + 浅蓝填充 + 各类思维模型图 + 数据看板 + 流程图 + 时间轴
- 适合人群：所有职场汇报需求；尤其适合作为"通用汇报素材库"长期使用

## 主题色
- 主色：`#1E3A5F`（深海蓝）
- 强调：`#2F6EBA`（亮蓝）
- 辅色：`#A6B5C8`（浅蓝灰）

## 字体
- 中文：思源黑体 / 微软雅黑

## 注意事项
- 这是"超大素材库"，**强烈建议按需挑页**而非全用；推荐组合 10-25 页做一份完整汇报。
- 大量页面顶部用 "X段内容页/" "工作汇报/" "对比关系页/" 等前缀，是版式标识，**使用时去掉前缀**。
- 信息密度极高，**字数严格控制**。
- 含部分原生 chart 对象（财务、销售、营销、HR 等数据页），可程序化更新；建议先用 python-pptx 检测。
- 与 report-savior、competition-speech、data-viz-deck、operations-deck、premium-corp 等同一作者系列，版式上有重叠；这是 5 套的**超集**版本。
